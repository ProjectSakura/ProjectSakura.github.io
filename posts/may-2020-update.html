Blog— Project Sakura

15 May 2020

Hello Everyone, Its May and we are here with another update of Project Sakura. This update took a while I know that but we added yet again many new changes that we are sure you are going to love.

The changes that you will be seeing in this release are,

  • May Security Patch r35.
  • All the new Lineage OS changes.
  • AOSPA style volume dialog.
  • LTE to 4G icon switch.
  • OOS style dismiss all to be used with R-Style nortification panel.
  • Material Ocean Theme.
  • Pitch black theme.
  • Redesigned battery settings from upcoming DOT OS Q.
  • Navigation handle length Improvements.
  • And few more improvements..

So this is all for the release. I really hope you guys will enjoy it and You can comment down to request any new feature for the next release.

Until then, Stay home stay safe.
