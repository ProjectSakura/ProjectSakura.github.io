Blog— Project Sakura

10 June 2020

It is better to be a young June bug than an old bird of paradise and So, we came up with a young new update. Following the monthly release cycles can be hectic sometimes *sigh*.

The changes that you should be seeing for june are,

  • June Security Patch r37.
  • As always updated with the latest Lineage source.
  • Quick QS brightness slider.
  • Faceunlock, with more devices actually adopting it.
  • Notification headsup customisations.
  • Support for proprietary fast charge detection(warp, dash, blah blah)
  • More and more clock faces.
  • Sensor blocking for some packages like facebook and whatsapp.
  • And some more fixes..

And last time three more devices joined us S8, s8+ and note 8. With this release Angler will be joining in.

With the next update we are coming up with few more demanded features :P, so please don't request for more feature this time.

Until then, Stay home stay safe.
