08 March 2021

Initial Release of Project Sakura 5.R

This spring feels more like summer, thanks to overheating kernels for causing Global Warming (Poor Joke :P). Well, We were so damn silent that people starting asking If we are alive anymore. Happy to tell you that, we are alive and we are updating every month like we always do. This month we are moving from version 4.R to 5.R and going into feature freeze, that means no more new feature requests will be taken and no more new features will be added anymore. So, please don't ask us to add anymore weird ass features.

Now see an ad.

So, What took us so long for this update? Here goes the changelog.

  • March Security Patch.
  • Better icon pack support in Trebuchet
  • App Drawer Opacity support in Trebuchet
  • Icon size support in Trebuchet
  • Moved Lineage Settings from System to SakuraConfig
  • Lockscreen weather
  • New IDE lockscreen clock
  • Data usage in QS Panel
  • Redesigned Volume Panel
  • Materialized Toasts
  • QS media player can be toggled ON and OFF
  • FaceUnlock animations
  • Notification sounds can be suppressed while screen is ON
  • Long ScreenShot support (Though it's kinda buggy)
  • And everything else that we cannot write here..

Huge ass update right? It took us so much work to get this all done during february. You guys better be really grateful to me this time. Also, we will try to increase the frequency of updates from this month and fix any remaining bugs, Please don't report about long screenshot coz we already know about it.

So, That was everything for this big release and see you soon with more new updates. Also, If you like how hard I work to bring you awesome and absolutely free software, you can go ahead and gift me some coffee, Buying me a coffee. Also I would like to thanks Somebody (That was the name) for buying me some coffee last month.

Well, this was it for this very update and if you find any bugs do report it to us with logcat coz Logcat or gtfo. As always do follow us on twitter @ProjectSakura_ coz we keep posting fun stuff on it ;)
