Okay good, so what we have new in this release..

  • Bubbles are fixed, we were constantly spammed for this one.
  • AOKP system animations are no more available in ROM.
  • For your ugly face that even satan can't recognize, Newer Faceunlock.
  • FOD icon picker and animations.
  • FOD animation effects.

Now getting to the main thing, If you are a developer with an FOD supported device and want to add support for FOD icon picker and animations you will need to do the following 

export EXTRA_FOD_ANIMATIONS=true
The above flag will build the standalone FOD animations resources needed for showing effects. In addition to that flag you will also need to have this config in your device tree, Link to commit. If you don't add it then FOD icon picker will be missing from Sakura Settings.
